sofware libre

La definicion de software libre estipula los criterios que se tienen que cumplir para que un programa sea considerado libre.

De vez en cuando modificamos esta definicion para clarificarla o para resolver problemas sobre cuestiones delicadas.

Mas abajo en esta pagina, en la seleccion Historial, Se puede consultar la lista de modificaciones que afectan la definicion de software libre.
